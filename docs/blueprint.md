# **App Name**: Ashfakur Rahman Asif - Digital Marketing Portfolio

## Core Features:

- Sticky Navbar: Sticky navigation bar with links to sections and a CV download button.
- Hero Section: Fullscreen hero section with title, subtitle, and CTAs to services and contact.
- Services Showcase: Cards showcasing offered marketing services.
- Portfolio Display: Case study-style cards for portfolio pieces, highlighting results.
- Social Links: Styled buttons linking to social media profiles.
- Firebase Contact Form: Contact form with Firebase integration.

## Style Guidelines:

- Primary color: Dark gray (#111827) for a professional base.
- Accent color: Teal green (#00B894) for highlights and actions.
- Background color: Light gray (#F9FAFB) for clean content presentation.
- Body and headline font: 'Poppins' (sans-serif) for a clean, modern aesthetic. Note: currently only Google Fonts are supported.
- Responsive design ensuring optimal viewing on all devices.
- Subtle scroll reveal and fade-in animations for smooth transitions.
