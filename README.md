# Ashfakur Rahman Asif - Digital Marketing Portfolio

This is a dynamic and interactive personal portfolio website built with Next.js and Firebase Studio. It showcases the skills, projects, and services of Ashfakur Rahman Asif, a Digital Marketing & Web Analytics Tracking Expert.

## Features

- **Dynamic Animations:** Built with `Framer Motion` and `AOS` for engaging scroll animations.
- **Interactive WebGL Logo:** A custom metallic liquid logo animation created with WebGL.
- **AI Chatbot:** An integrated chatbot powered by Google's Gemini model via Genkit, capable of answering questions about Ashfakur's services.
- **Fully Responsive Design:** A beautiful and functional layout that works on all devices.
- **Contact Form:** A functional contact form using Resend for email delivery.

## Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **UI Components:** [ShadCN UI](https://ui.shadcn.com/)
- **AI Integration:** [Genkit (Google AI)](https://firebase.google.com/docs/genkit)
- **Animations:** [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/) & [Three.js](https://threejs.org/) for WebGL.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js (v18 or later)
- npm or yarn

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your_username/your_repository.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the development server
   ```sh
   npm run dev
   ```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.
